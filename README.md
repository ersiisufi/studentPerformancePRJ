# Student Performance Prediction

## Problem
Predict student exam performance based on study habits and past data.

## Approach
- Data cleaning
- Linear regression model
- Evaluation using MSE and R²

## Results
- MSE: 28
- R²: 0.88

## How to Run

### Train model
python train_pipeline.py

### Predict
python predict_pipeline.py
